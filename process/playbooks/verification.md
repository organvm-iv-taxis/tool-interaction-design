# Verification Playbook

## Purpose
Prove behavior, reliability, and contract compliance.

## Entry Criteria
- Implementation is complete.
- Required tests are added or updated.

## Procedure
1. Run unit/integration tests.
2. Run schema validations.
3. Run policy simulation and observability checks.
4. Capture CI and demo evidence URLs.
5. Update work item Definition of Done fields.

## Exit Criteria
- All required gates pass.
- Demo evidence exists.

## Outputs
- Completed DoD block with evidence links.
