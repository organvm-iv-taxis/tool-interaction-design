# Recovery Drill Checklist

- [ ] Select scenario (schema break, migration failure, policy breach, service regression).
- [ ] Assign incident commander and scribe.
- [ ] Start timer and declare severity.
- [ ] Execute rollback/hotfix from runbook.
- [ ] Validate system health after mitigation.
- [ ] Record time-to-detect and time-to-recover.
- [ ] Capture gaps in tooling/playbooks.
- [ ] File corrective work items with owners and due dates.
